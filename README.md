# Password Strength Checker (Vanilla JS)

A browser-based password strength checker with instant feedback and common security rules.

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## Run Locally
Open `src/index.html` in your browser.

## Project Structure
- `src/index.html` — UI
- `src/styles.css` — styling
- `src/app.js` — logic

## What I practiced
- Real-time input handling and rendering
- Regex-based checks and small utility functions
- Presenting clear UX feedback without frameworks
